## What is fz-connect?

fz-connect is a FiveM-focused bot built to surface one thing clearly: how many players are connected to your server right now. It is aimed at server owners and community managers who need a fast, visible status indicator without manually checking the server.

The project fits into common FiveM admin workflows where player activity changes frequently and live visibility is useful. By keeping the connected-player total up to date, it gives you a simple status readout that can feed dashboards, bots, or other server-side tools.
